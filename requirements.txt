# DESIGN NOTE ON THE SDK (read this before installing):
#
# You asked for the "unified Polymarket Python SDK (py-sdk)". That package is
# real (installed as `polymarket-client`, imported as `polymarket`) but it is
# currently published with a BETA badge, its public API is explicitly described
# as unstable, and — critically — its README only documents *public market
# reads* (get_market). It does NOT yet publicly document order placement,
# cancellation, or order-book streaming, which are exactly the surfaces a
# money-handling bot lives or dies on.
#
# For that reason this codebase is built on `py-clob-client`, the production
# CLOB SDK (currently ~v0.34.x, the one Polymarket's own docs use for order
# placement). It is wrapped behind a thin adapter in `exchange.py`, so when
# `polymarket-client` stabilises its trading API you can swap the adapter
# internals without touching strategy/risk/main. See README.md.
#
# ----------------------------------------------------------------------------
